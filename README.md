# Genesik

### Huge symbolic music using dataset
* Data: MIDI files
* Labels: Metadata (style, compositor, instruments, etc.) to steer generation or do supervised learning

##### Library
* to_matrix
* to_midi
* download_dataset
* read_data
* data_augmentation (transposition, rythm, etc.)
* change_mode

##### Representation
* MIDI
* Binary matrix
